## Installation Guide
1. Create new Database with name `InovApp` or other
2. Create new MySQL user (e.g. `InovRoot@localhost`)<br />
     Grant access to the user `grant all on InovApp.* to InovRoot@localhost`
3. Run `cp .env.example .env`
4. Set your `.env`:
     - `APP_NAME` with Application Name
     - `APP_ENV` with `production`
     - `APP_DEBUG` with `false`
     - `APP_URL` with current domain that have been determined
     - `DB_DATABASE`,`DB_USERNAME`, `DB_PASSWORD`, with database configuration above

5. Run `composer install`
6. Run `php artisan key:generate`
7. Run `php artisan config:cache`
8. Run `composer require laravel/passport`
9. Run `php artisan migrate`
10. Run `php artisan db:seed` <br/>
		note: step 9 and 10 can be replaced by `php artisan migrate:refresh --seeds`
11. Run `php artisan passport:install`
12. Run `sudo npm run production`<br/>


13. Please make sure that your web server already configured to `public/` folder<br/>
    Can be done by changing apache's documentRoot to `home/<path>/<to>/<project>/public/`
14. To start server, run `php artisan serve`<br/>

## Laravel DB Migration Guide

1. Create new scheme migration script with `php artisan make:migration "Create something table"`
2. Edit the script under /database/migrations/ 
3. Run `php artisan migrate` to apply new scheme, or if necesssary, 
4. Run `php artisan migrate:reset` to rollback last update before run the `migrate`
   
     Check [full-documentation](https://laravel.com/docs/5.8/migrations) and DB scheme [cheatsheet](https://laravel.com/docs/5.8/migrations#creating-tables)
